js-tagger
=========

Small pure javascript to transform a input field into a tag field


include the javascript:

    <script src=js/js-tagger.js></script>
    
have and input:

    <input id="my_tag_field" value="test, test2" />
    
Activate the plugin:

    <script>
        tagger = new JSTagger("my_tag_field");
    </script>
